Kalshi R Package

R client for the Kalshi prediction market API.

Project Status

Version: 0.1.0 Status: Functional, ready for use

Package Structure

R/
├── auth.R       # Authentication (RSA-PSS signing via system openssl)
├── utils.R      # Internal helpers (request building, pagination, date parsing)
├── markets.R    # Market data endpoints
├── events.R     # Events and series endpoints
├── portfolio.R  # Portfolio endpoints (balance, positions, orders, fills)
├── orders.R     # Trading endpoints (create, cancel, amend orders)
├── history.R    # Historical data analysis functions

Key Design Decisions

Authentication

  • Uses system openssl binary for RSA-PSS signing (R’s openssl package doesn’t expose PSS padding)
  • Private key path stored in internal environment, never read into R memory
  • API Key ID stored in .Renviron as KALSHI_API

API Base URL

  • Production: https://api.elections.kalshi.com
  • Demo: https://demo-api.elections.kalshi.com

Data Returns

  • All list endpoints return tibbles
  • Pagination handled automatically via kalshi_paginate()
  • Dates can be passed as strings (“2026-02-07”), Date objects, or Unix timestamps

Dependencies

  • httr2 - HTTP requests
  • cli - User-facing messages
  • purrr - Functional programming
  • tibble - Data frames
  • openssl - Key generation and base64 encoding
  • usethis - Environment file editing

Usage Examples

Setup

library(kalshi)

# First time: generate key pair
generate_kalshi_keypair("kalshi-key.key")
# Upload public key to Kalshi dashboard, get Key ID

# Add to .Renviron: KALSHI_API=your-key-id
usethis::edit_r_environ()

# Each session: set key path
kalshi_auth_setup("kalshi-key.key")

Get Weather Market History

# Get results for NYC temperature on a specific date
results <- get_event_results(series_ticker = "KXHIGHNY", date = "2026-02-07")

# See price evolution (hourly candles)
candles <- get_candlesticks(
  ticker = "KXHIGHNY-26FEB07-B26.5",
  series_ticker = "KXHIGHNY",
  start = "2026-02-06",
  end = "2026-02-08",
  period_interval = 60
)

# Get all historical results for a series
history <- get_series_history("KXHIGHNY")

Trading

# Place a limit order
order <- create_order(
  ticker = "KXHIGHNY-26FEB08-B26.5",
  side = "yes",
  action = "buy",
  count = 10,
  yes_price = 45
)

# Cancel it
cancel_order(order$order_id)
